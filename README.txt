Glori Budget Manager

Run:
1. open Terminal in this folder
2. npm install
3. npm start
4. open http://localhost:3000

Notes:
- data/budget.xlsx will be created automatically
- project codes auto-generate like GB-0001
- Download Excel button exports the current workbook
- If your old budget.xlsx still contains the old stock structure, this revised server recreates the Projects sheet header for the new budget system
